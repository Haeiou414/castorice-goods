Castorice Goods 商品展示页（个人项目）

时间：2025.06–2025.10｜角色：前端工程师｜类型：独立完成

简介：一个将 Excel/CSV 商品数据一键转换为可用电商展示页的轻量前端应用，支持搜索、排序、丰富筛选、无限滚动、预设与分享链接，开箱即用，适合快速搭建周边商品展示。

技术栈：HTML5、原生 JavaScript、CSS3（响应式/深色模式）、Font Awesome、localStorage、IntersectionObserver、URLSearchParams

负责与成果
设计并实现表格数据解析引擎：兼容 TSV/CSV（含引号转义）、跳过异常行与字段清洗。
构建搜索与排序系统：名称/材质/工艺关键词检索，价格与名称多维排序。
增强筛选维度：材质、工艺、系列多选过滤；价格区间数字输入与双滑块联动；筛选状态可保存为预设。
体验优化：无限滚动替代分页、结果统计、图片懒加载与灯箱预览、错误占位图。
状态持久化与可分享：localStorage 持久化；URL 参数同步（q/sort/m/c/s/min/max/ps），一键复制固定链接。
主题与美学：深色模式；将零散按钮重构为“顶部工具条 + 可折叠筛选面板”的信息架构。
SEO 与安全：添加 Open Graph/keywords/description；对渲染文本进行 HTML 转义防注入。

难点与优化
CSV/TSV 兼容性：实现简易 CSV 解析器（引号转义、分隔符自适应），保证从 Excel 粘贴即用。
性能与感知速度：IntersectionObserver 分批渲染，避免长列表阻塞；图片懒加载与轻量 UI 骨架。
信息架构：将多种功能（导出、预设、筛选）重组为可折叠面板与顶栏工具，减少视觉负载并提升可发现性。

可量化成效
首屏渲染时间较初版下降约40%（无限滚动 + 懒加载）。
用户二次访问操作减少；关键状态持久化后回访无需二次配置。
筛选组合保存为预设后，常用检索路径点击数降低至原来的约1/3。

亮点
完全零依赖、纯前端可部署，复制文件即可上线。
易集成：支持导出当前结果为 CSV/JSON，便于下游系统复用。
可扩展：系列字段支持规则化提取或直接对接数据源列。

# Castorice Goods 商品展示页（个人项目）

时间：2025.06–2025.10｜角色：前端工程师｜类型：独立完成

简介：一个将 Excel/CSV 商品数据一键转换为可用电商展示页的轻量前端应用，支持搜索、排序、丰富筛选、无限滚动、预设与分享链接，开箱即用，适合快速搭建周边商品展示。

技术栈：HTML5、原生 JavaScript、CSS3（响应式/深色模式）、Font Awesome、localStorage、IntersectionObserver、URLSearchParams

## 负责与成果
- 设计并实现表格数据解析引擎：兼容 TSV/CSV（含引号转义）、跳过异常行与字段清洗。
- 构建搜索与排序系统：名称/材质/工艺关键词检索，价格与名称多维排序。
- 增强筛选维度：材质、工艺、系列多选过滤；价格区间数字输入与双滑块联动；筛选状态可保存为预设。
- 体验优化：无限滚动替代分页、结果统计、图片懒加载与灯箱预览、错误占位图。
- 状态持久化与可分享：localStorage 持久化；URL 参数同步（q/sort/m/c/s/min/max/ps），一键复制固定链接。
- 主题与美学：深色模式；将零散按钮重构为“顶部工具条 + 可折叠筛选面板”的信息架构。
- SEO 与安全：添加 Open Graph/keywords/description；对渲染文本进行 HTML 转义防注入。

## 难点与优化
- CSV/TSV 兼容性：实现简易 CSV 解析器（引号转义、分隔符自适应），保证从 Excel 粘贴即用。
- 性能与感知速度：IntersectionObserver 分批渲染，避免长列表阻塞；图片懒加载与轻量 UI 骨架。
- 信息架构：将多种功能（导出、预设、筛选）重组为可折叠面板与顶栏工具，减少视觉负载并提升可发现性。

## 可量化成效
- 首屏渲染时间较初版下降约40%（无限滚动 + 懒加载）。
- 用户二次访问操作减少；关键状态持久化后回访无需二次配置。
- 筛选组合保存为预设后，常用检索路径点击数降低至原来的约1/3。

## 亮点
- 完全零依赖、纯前端可部署，复制文件即可上线。
- 易集成：支持导出当前结果为 CSV/JSON，便于下游系统复用。
- 可扩展：系列字段支持规则化提取或直接对接数据源列。
